# Report Builder en Adobe Learning Manager

Crea y descarga informes personalizados con las columnas, los filtros y los datos que elijas, sin tener que procesar el contenido con ninguna herramienta.

El Report Builder de Adobe Learning Manager proporciona a los administradores un lienzo de informes de autoservicio para crear exactamente los informes que necesitan. En lugar de descargar un informe fijo y modificarlo en una herramienta de hoja de cálculo, seleccione las columnas que desee, aplique filtros y descargue un resultado limpio, todo desde un único lugar.

## ¿Por qué se presentó Report Builder?

Se han solucionado los informes existentes en Adobe Learning Manager. Cada informe tiene una lista establecida de columnas que no se pueden cambiar y las opciones de filtrado son limitadas. Los administradores que necesitaban resultados personalizados tenían que descargar varios informes, unirse a ellos en Excel, aplicar filtros manualmente y repetir el proceso cada semana o mes.

Report Builder elimina ese ciclo. Configure un informe una vez, guárdelo y descargue datos nuevos siempre que los necesite, sin necesidad de cambiar su forma.

El Report Builder resuelve tres problemas específicos:

* **Columnas fijas:** Los informes existentes incluyen muchas columnas que quizá no necesite y excluye las que sí necesita. Report Builder le permite seleccionar solo los campos relevantes para su caso de uso.
* **Filtrado limitado:** Ahora puede filtrar en cualquier campo compatible. Por ejemplo, incluyendo la fecha de inscripción, la fecha de finalización, los campos activos y las etiquetas de catálogo, no solo el estado.
* **Fuentes de datos incoherentes:** Los informes existentes se pueden extraer de la interfaz de usuario, la API de trabajos o los conectores, que pueden devolver datos ligeramente diferentes según la temporización. Report Builder extrae de un único origen de datos coherente, de modo que cada descarga refleja los mismos datos subyacentes.

## Cómo encaja el Report Builder con los informes existentes

Report Builder no reemplaza los informes fijos existentes en Adobe Learning Manager. Ambos están disponibles. Utilice los informes existentes para los resultados estándar en los que ya confía. Utilice Report Builder cuando necesite columnas personalizadas, filtros específicos o datos que requieran procesamiento posterior.

## Dónde encontrar Report Builder

Report Builder está disponible en la página principal del administrador. Seleccione **Informes** y, a continuación, seleccione **Report Builder**. Verá dos fichas: **Plantillas** e **informes**.
![](assets/report-builder-001.png)

## Quién puede usar Report Builder

Report Builder está disponible para los administradores.

>[!NOTE]
>
>El Report Builder debe estar habilitado para las cuentas. Póngase en contacto con el equipo de cuentas de Adobe o el CSM si no aparece en la vista de administrador.

## Prácticas recomendadas

* Comience con una plantilla si es nuevo en Report Builder. Las plantillas están prediseñadas para los casos de uso más comunes y funcionan de inmediato.
* Guarde todos los informes configurados antes de descargarlos. Los informes guardados se pueden descargar de nuevo más adelante sin necesidad de volver a configurarlos.
