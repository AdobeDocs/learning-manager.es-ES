# Duplicación y reutilización de un informe en Report Builder

Al duplicar un informe se copian todas sus columnas, alias, configuración de agrupar por, agregaciones, filtros y ordenación en un nuevo informe editable. Utilice esta opción cuando desee crear una versión de un informe para un catálogo, grupo de usuarios o período de tiempo diferente.

## Duplicar un informe

1. Abra el informe que desea duplicar.
2. Seleccione **Acciones** > **Duplicar**. Se abre una copia del informe en modo de edición con el nombre &quot;[nombre del informe original]-Copiar&quot;. Por ejemplo, Vista de rendimiento estructurada entre gestores - Copiar
3. Introduzca un nombre nuevo para el informe.
4. Realice los cambios que desee, por ejemplo, actualizar filtros, ajustar columnas o cambiar la ordenación.
5. Seleccione **Guardar informe**.

El informe duplicado se guarda en la pestaña **Informes** y es independiente del original.

## Duplicar una plantilla

También puede duplicar plantillas desde la pestaña **Plantillas**. Al duplicar una plantilla, se crea un nuevo informe editable en la pestaña **Informes**. La plantilla original permanece sin cambios.

1. Seleccione la ficha **Plantillas**.
2. Busque la plantilla que desea copiar.
3. Seleccione **Duplicar**.
   ![](assets/image049.png)
4. Escriba un nombre, realice ajustes y seleccione **Guardar informe**.

## Prácticas recomendadas

* Duplica un informe bien probado como base cuando necesites variaciones para diferentes equipos, catálogos o periodos de tiempo. Esto es más rápido que construir desde cero y garantiza una estructura coherente.
* Cambie el nombre de la copia inmediatamente para reflejar qué la diferencia del original, por ejemplo, &quot;Cumplimiento por VP - APAC&quot; en lugar de &quot;Copia del informe de cumplimiento&quot;.
* Después de duplicar una plantilla, actualice el nombre antes de realizar cualquier otro cambio para que el informe sea identificable en la pestaña **Informes**.
