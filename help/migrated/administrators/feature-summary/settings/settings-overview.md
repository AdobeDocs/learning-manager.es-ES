# Configuración

## Información general

Adobe Learning Manager proporciona configuraciones de administrador detalladas que le permiten configurar y personalizar su sistema de gestión de aprendizaje para satisfacer las necesidades específicas de su organización. La configuración se organiza en varias categorías clave, cada una de las cuales cumple distintas funciones administrativas.

### Configuración básica

* Esta sección permite a los administradores configurar los detalles esenciales de la organización, incluidos el país, la región, la zona horaria o la configuración regional.
* Configurar mecanismos de comentarios y sistemas de clasificación para objetos de aprendizaje.
* Otras configuraciones generales que afectan al comportamiento y uso generales de la plataforma.

Consulte [configuración](/help/migrated/administrators/feature-summary/settings/basic-settings.md) en Adobe Learning Manager para obtener más información.

### Configuración de integración

* Configure las opciones de autenticación, como Adobe ID e Inicio de sesión único (SSO).
* Gestiona integraciones con sistemas de datos externos para el aprovisionamiento de usuarios, los catálogos de cursos y la sincronización de datos de informes.
* Configure y administre conexiones con otras cuentas de Learning Manager para compartir contenido e iniciativas de aprendizaje colaborativo.

Consulte la configuración de Adobe Learning Manager para obtener más información.

### Configuración avanzada

* Crea y gestiona etiquetas personalizadas para organizar y categorizar el contenido de aprendizaje, lo que facilita a los alumnos la búsqueda de materiales relevantes.
* Configura y gestiona ubicaciones de clases físicas para sesiones de formación dirigidas por instructores, incluidos los detalles de capacidad, equipo y programación.
* Configurar paneles de cumplimiento y éxito de grupo.

Consulte la configuración de Adobe Learning Manager para obtener más información.
